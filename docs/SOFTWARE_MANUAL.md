## 二、软件功能介绍

### 2.1 软件主要功能

本软件是一款轻量、稳定、开源的 Android 短信转发应用，主要功能包括：

1. **多通道短信转发**
   - 支持企业微信 Webhook 转发
   - 支持钉钉 Webhook 转发
   - 支持飞书 Webhook 转发
   - 支持通用 Webhook 转发（自定义格式）

2. **关键词过滤**
   - 灵活配置关键词规则
   - 空关键词表示转发全部
   - 支持配置多个关键词
   - 支持配置目标转发通道

3. **验证码自动提取**
   - 自动识别短信中的验证码
   - 支持中英文关键词（验证码、校验码、verification 等）
   - 支持 4-8 位数字验证码
   - 突出显示验证码，方便复制

4. **双卡设备支持**
   - 支持识别接收短信的 SIM 卡号码
   - 双卡设备支持显示 SIM1/SIM2 的本机号码
   - 无法自动获取时支持手动输入本机号码

5. **消息格式配置**
   - 可配置是否显示本机号码
   - 可配置是否显示发送者号码
   - 可配置是否突出显示验证码

6. **消息去重**
   - 5秒内相同内容自动去重
   - 避免重复转发

7. **智能重试**
   - 失败消息最多重试 3 次
   - 指数退避策略（2s/4s/6s）
   - 网络恢复自动重试

8. **持久化存储**
   - 失败消息保存本地
   - 应用重启后自动重试
   - 网络恢复后自动重试

9. **前台服务常驻**
   - 提高在厂商定制 ROM（如 HyperOS）上的存活率
   - 显示常驻通知

10. **运行日志记录**
    - 完整日志保存
    - 方便排查问题

11. **电池优化引导**
    - 引导用户加入电池优化白名单
    - 减少系统杀死概率

12. **开机自启**
    - 设备启动后自动运行
    - 可选配置

### 2.2 软件特点

1. **轻量级**：应用体积小，安装包约 11-12MB

## 五、软件使用说明

### 5.1 首次使用

1. **打开应用**
   - 点击应用图标打开应用

2. **授予权限**
   - 授予短信接收权限
   - 授予通知权限（可选但推荐）
   - 授予开机自启权限（可选）

3. **配置转发通道**
   - 进入"转发通道"标签页
   - 点击"添加通道"
   - 选择通道类型（企业微信/钉钉/飞书/Webhook）
   - 填写通道名称和 Webhook 地址
   - 点击"保存"

4. **配置关键词规则**
   - 进入"关键词"标签页
   - 输入关键词（留空表示转发全部）
   - 选择目标转发通道
   - 点击"添加配置"

5. **配置 SIM 卡号码**
   - 进入"设置"标签页
   - 查看 SIM 卡信息
   - 如无法自动获取，可手动输入

6. **开启转发服务**
   - 进入"首页"标签页
   - 打开"短信转发"开关
   - 配置"开机自启动"（可选）

### 5.2 日常使用

1. **接收短信**
   - 应用自动监听收到的短信
   - 根据关键词规则匹配
   - 自动转发到配置的通道

2. **查看日志**
   - 进入"日志"标签页
   - 查看转发记录
   - 查看失败消息
   - 手动重试失败消息

3. **测试规则**
   - 进入"首页"标签页
   - 点击"测试规则"
   - 输入测试短信内容
   - 查看匹配结果

### 5.3 消息格式

转发的消息格式如下：

```
验证码: 123456
本机: 13800138000
来自: 106900000000

【某某公司】您的验证码是 123456，请在 5 分钟内完成验证。
```

消息格式可配置，支持：
- 是否显示验证码
- 是否显示本机号码
- 是否显示发送者号码

## 六、软件技术特点

### 6.1 技术架构

- **开发语言**：Kotlin
- **UI 框架**：Jetpack Compose
- **异步处理**：Kotlin Coroutines
- **网络请求**：OkHttp
- **JSON 解析**：org.json
- **最低 SDK**：API 21 (Android 5.0)
- **目标 SDK**：API 34 (Android 14)

### 6.2 核心技术点

1. **短信监听**
   - 使用 BroadcastReceiver 监听短信广播
   - 高优先级（999）确保优先接收

2. **双卡识别**
   - 使用 SubscriptionManager 获取 SIM 卡信息
   - 从短信 Intent 中提取 subscriptionId
   - 支持手动输入 SIM 卡号码

3. **验证码提取**
   - 正则表达式匹配
   - 支持中英文关键词
   - 智能位置识别

4. **消息去重**
   - 时间窗口去重（5秒）
   - 内容哈希对比

5. **重试机制**
   - 指数退避策略
   - 本地持久化存储
   - 网络状态监听

6. **前台服务**
   - 提高应用存活率
   - 常驻通知显示
   - 支持停止服务

## 十、软件使用注意事项

1. **权限说明**
   - 本软件需要短信接收权限才能正常工作
   - 建议授予通知权限以提高应用存活率
   - 开机自启权限为可选权限

2. **网络说明**
   - 本软件需要网络连接才能转发短信
   - 仅通过 HTTPS 协议转发，确保数据安全
   - 短信内容仅转发到用户配置的地址

3. **隐私保护**
   - 短信内容仅在本地临时存储
   - 不会上传到开发者服务器
   - 敏感配置信息加密存储

4. **电池优化**
   - 建议将应用加入电池优化白名单
   - 可减少系统杀死概率
   - 提高转发成功率
