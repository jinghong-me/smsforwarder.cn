# 隐私政策

**最后更新日期：2026年4月6日**

## 概述

感谢您使用短信转发助手（以下简称"本应用"）。我们非常重视您的隐私保护，并将严格按照本隐私政策收集、使用、存储和保护您的个人信息。

本应用是由华昊科技有限公司（以下简称"我们"）开发的短信转发工具应用。本隐私政策旨在向您说明我们如何处理您的个人信息，请您仔细阅读并了解本隐私政策的全部内容。

## 核心原则

我们的承诺：

- ✅ **不上云**：所有数据都在您的手机本地处理，不会上传到我们的服务器
- ✅ **不收集**：不会收集您的个人信息、短信内容等敏感数据
- ✅ **不追踪**：不集成任何统计、分析或广告 SDK
- ✅ **完全可控**：所有权限和数据都由您自己掌控

## 信息收集与使用

### 短信内容

- **用途**：仅在您的手机本地用于匹配关键词规则和执行转发
- **存储**：不会保存到任何服务器，仅在转发时临时处理

**重要**：唯一会发送短信内容的情况是**您主动配置了 Webhook 转发目标**（如企业微信、钉钉、飞书或自定义 Webhook），应用会将短信直接发送到**您指定的目标**，不会经过我们的服务器。

### 配置信息

您设置的转发通道、关键词规则等配置信息保存在您手机的本地存储中，不会上传。

### 转发日志

应用会在本地记录转发历史（最多200条），方便您查看和调试，这些日志仅存储在您的手机上。

## 权限说明

本应用需要以下权限：

| 权限 | 用途 | 是否必需 |
|------|------|----------|
| RECEIVE_SMS | 监听设备收到的短信，用于执行转发功能 | 是 |
| READ_PHONE_STATE | 用于识别双卡手机的 SIM 卡信息和获取本机号码 | 否（可选） |
| POST_NOTIFICATIONS | 显示前台服务通知，让您知道服务正在运行 | 是 |
| INTERNET | 仅用于转发到您配置的 Webhook | 是 |
| RECEIVE_BOOT_COMPLETED | 让应用在开机后自动启动转发服务 | 否（可选） |
| FOREGROUND_SERVICE | 保持应用在后台稳定运行 | 是 |
| FOREGROUND_SERVICE_REMOTE_MESSAGING | 前台服务类型 | 是 |
| REQUEST_IGNORE_BATTERY_OPTIMIZATIONS | 防止系统杀死后台服务 | 否（可选） |
| ACCESS_NETWORK_STATE | 检测网络连接状态 | 是 |

所有权限都需要您**主动授权**，您可以随时在系统设置中撤销。

## 数据存储

### 本地存储

所有数据都存储在您手机的私有目录中，包括：

- 转发通道和关键词配置
- 转发历史日志
- 应用设置

### 服务器存储

**我们没有服务器存储您的数据！**

应用是纯本地运行的工具，我们不收集、不存储、不上传任何用户数据。

## 您的权利

您拥有以下权利：

1. **查看数据**：可以在应用内查看所有转发日志
2. **删除数据**：可以在应用内清空日志，或卸载应用删除所有数据
3. **控制权限**：可以在系统设置中随时授予或撤销权限

## 第三方服务

**关于转发目标**：

如果您配置了 Webhook 或其他第三方服务作为转发目标，短信内容会发送到该第三方。请您谨慎选择转发目标，并确保了解其隐私政策。我们不对第三方的数据处理负责。

### 第三方 SDK

当前版本未集成任何第三方 SDK（包括统计、广告、崩溃分析等）。

我们可能会不时更新本隐私政策。重大变更时，我们会通过应用内通知或其他方式告知您。建议您定期查看本政策以了解最新信息。
