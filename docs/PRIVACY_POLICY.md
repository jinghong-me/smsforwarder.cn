# 隐私政策

**最后更新日期：2026年4月5日**

## 引言

感谢您使用短信转发助手（以下简称"本应用"）。我们非常重视您的隐私保护，并将严格按照本隐私政策收集、使用、存储和保护您的个人信息。

本应用是由华昊科技有限公司（以下简称"我们"）开发的短信转发工具应用。本隐私政策旨在向您说明我们如何处理您的个人信息，请您仔细阅读并了解本隐私政策的全部内容。

## 适用范围

本隐私政策适用于短信转发助手应用及其相关服务。

## 信息收集和使用

### 1. 短信内容

- **收集方式**：通过 Android 系统短信接收权限获取
- **收集目的**：用于转发短信到您配置的目标地址
- **存储方式**：仅在本地临时存储，用于转发失败重试
- **上传情况**：仅上传到您配置的 Webhook 地址，不会上传到我们的服务器
- **保留期限**：转发成功后立即删除，失败消息最长保留7天

### 2. 电话号码信息

- **收集方式**：通过 READ_PHONE_STATE 权限获取 SIM 卡信息
- **收集目的**：用于在转发短信时显示接收短信的 SIM 卡号码
- **存储方式**：本地 SharedPreferences 存储
- **上传情况**：不上传任何号码到我们的服务器

### 3. 网络信息

- **收集方式**：通过 ACCESS_NETWORK_STATE 权限获取
- **收集目的**：用于监听网络状态变化，网络恢复时自动重试转发
- **存储方式**：本地临时存储
- **上传情况**：不上传

### 4. 通知权限

- **收集方式**：通过 POST_NOTIFICATIONS 权限
- **收集目的**：用于显示常驻通知，提高应用存活率
- **存储方式**：不存储
- **上传情况**：不上传

## 权限说明

本应用需要以下权限：

| 权限 | 用途 | 是否必需 |
|------|------|----------|
| RECEIVE_SMS | 接收短信 | 是 |
| INTERNET | 转发短信到 Webhook 地址 | 是 |
| FOREGROUND_SERVICE | 前台服务，提高应用存活率 | 是 |
| RECEIVE_BOOT_COMPLETED | 开机自启动 | 否（可选） |
| POST_NOTIFICATIONS | 显示常驻通知 | 是 |
| FOREGROUND_SERVICE_REMOTE_MESSAGING | 前台服务类型 | 是 |
| REQUEST_IGNORE_BATTERY_OPTIMIZATIONS | 电池优化白名单，提高存活率 | 否（可选） |
| ACCESS_NETWORK_STATE | 监听网络状态 | 是 |
| READ_PHONE_STATE | 获取 SIM 卡信息 | 否（可选） |

## 信息共享

我们不会将您的个人信息共享给任何第三方，除非：

1. **获得您的明确同意**：在获得您的明确同意后，我们才会共享您的信息
2. **法律法规要求**：根据法律法规、政府主管部门的要求或强制性的行政命令
3. **您配置的 Webhook 地址**：短信内容仅转发到您自己配置的 Webhook 地址

## 信息安全

我们采取以下安全措施保护您的信息：

1. **本地存储加密**：敏感配置信息加密存储
2. **HTTPS 传输**：仅支持 HTTPS 协议转发短信
3. **最小权限原则**：仅申请必要的权限
4. **定期清理**：定期清理本地临时存储的失败消息
